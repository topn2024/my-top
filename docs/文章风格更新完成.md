## 📝 新风格特点

### ✅ 采用的格式：
- **自然段落**：用空行分隔，有呼吸感
- **重点标注**：用【】或「」标记重要内容
- **数字序号**：1、2、3、或一、二、三、
- **emoji表情**：✨ 🎯 💡 🔥 适当使用
- **口语化**：说实话、不得不说、这个真的...
- **像聊天**：通俗易懂的大白话

### ❌ 不再使用：
- Markdown标题：`# ## ###`
- Markdown列表：`- * +`
- Markdown加粗：`** **`
- Markdown代码块：` ``` ``` `
- Markdown引用：`>`
- 过于格式化的排版

## 🎯 Prompt核心要点

### 内容要求
- 文章长度800-1200字
- 用通俗易懂的大白话
- 像跟朋友聊天一样
- 可以用实际案例、数据说明
- 标题要吸引眼球

### 写作风格（重要！）
```
✍️ 写作风格：
- 使用自然的段落分隔，不要用Markdown标记
- 可以用空行分段，让文章有呼吸感
- 重点内容用「」或【】标注，不要用加粗**
- 数字、专有名词直接写，不要加引号
- 适当使用emoji表情符号 ✨ 🎯 💡
- 语气自然，像在知乎、微博发帖
- 可以用口语化表达
```

### 绝对不要使用
```
❌ 绝对不要使用：
- Markdown标题格式：# ## ###
- Markdown列表：- * +
- Markdown加粗：** **
- Markdown代码块：``` ```
- Markdown引用：>
- 过于生硬的格式化排版
```

### 推荐使用
```
✅ 推荐使用：
- 自然的段落（用空行分隔）
- 【标题】或「重点」这样的标注
- emoji表情符号 ✨ 🎯 💡 🔥
- 数字序号：1、2、3、或一、二、三、
- 像说话一样的表达方式
```
